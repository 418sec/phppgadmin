New features in version 1.08
  There are mainly three updates between version 1.05 and 1.08:
  • Click on document icons opens document
    When you click on a document icon, or on the document title, the corresponding document is opened.
  • Totaly expanded, not collapsable, list is provided by a special function.
    It is now possible to get a totaly expanded 'listing' on all sub-items to s specific item.
  • Easier customization thought separate configuretion file.
    You don't has to edit the main script file any more to be able to customize explorerMenu, all the settings are in a separate file.
How to install
  To be able to use explorerMenu on your site, your site must be a framed site, i.e. it must use frames, and one frame must be reserved for the menu.
  To install explorerMenu on your site, first download the explorereMenu package, which contains the script file, an example configuration file and and example menu file. After downloading the file, gunzip and untar it (Or use WinZIP on a windosze-platform) in the web site root directory. Then edit your index.html to conform to the style of example.html. Edit scripts/config.js to customize the look of the menu, and scripts/menu.js to customize its contents.
